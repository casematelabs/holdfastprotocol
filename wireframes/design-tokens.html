01 — Protocol Identity

Protocol Accent System

Both protocols share background, border, and typography systems. Accent tokens are the primary differentiator. Holdfast Protocol uses a clean digital blue (chain-native trust). Hardline uses a deep violet (governance authority). Both read as institutional — neither hostile, neither casual.

02 — Shared System

Background Hierarchy

Four levels of depth create layering without color variation. All elevation is dark, cold, precise. Used identically across Holdfast Protocol and Hardline surfaces.

--bg-page
#0D1117
Base canvas. Page background, outermost surface.
--bg-card
#141B27
Primary card surface. First layer above page.
--bg-card-raised
#18202E
Elevated card or inner panel. Modals, drawers, nested sections.
--bg-hover
#1A2438
Hover state surface. Rows, list items, interactive tiles.

03 — Shared System

Border System

Three border states: default (structural), hover (interactive response), focus (input active state). Focus border derives from the protocol accent — override in Hardline Protocol contexts.

--border
#1E2D42
Default structural edge. Cards, tables, dividers.
--border-hover
#2D4060
Interactive hover. Brightens on pointer interaction.
--border-focus
rgba(45, 140, 255, 0.50)
Active input focus ring. Accent-derived.

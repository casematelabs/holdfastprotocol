Grid & Sizing
All icons live on a 24×24px grid with 2px internal padding. Scale at 16px, 20px, or 24px — never between. The 1.5px stroke weight is maintained at all standard sizes.
Stroke Style
Square linecaps (butt cap), miter joins. No fills on path interiors — only solid dots or nodes as fill accent. Opacity variants (50%) used for secondary paths only.
Colour Usage
Icons default to currentColor. Use --text-secondary at rest, --accent on hover/active states.

Getting started with the Holdfast Protocol SDK

The Holdfast Protocol SDK exposes a typed TypeScript interface to the on-chain Anchor program. You can lock funds into an escrow account, stake a beneficiary, and manage the full escrow lifecycle without writing raw instruction data. This guide covers installation, wallet integration, and your first escrow transaction on devnet.

07 Motion & Animation

Motion Principles

Motion is purposeful and subtle. It communicates state changes — not decoration. Entrances are fast (150–250ms). Persistent animations (pulses, shimmers) are low-energy. Nothing bounces. Nothing slides in from off-screen.

typescript Copy 
import { Holdfast ProtocolClient } from '@holdfastprotocol/sdk';

const client = new Holdfast ProtocolClient({ cluster: 'devnet' });

// Lock funds into escrow with a 30-day deadline
const tx = await client.escrow.lockEscrow({
  depositor:         wallet.publicKey,
  beneficiary:       beneficiaryPubkey,
  amount:            1_500_000_000, // 1.5 SOL in lamports
  deadlineSecs:      2_592_000,    // 30 days
});

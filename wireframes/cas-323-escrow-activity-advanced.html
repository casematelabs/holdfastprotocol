01 — AnalyticsSummaryBar
Analytics Summary
Four stat cards above the pact table. Each card shows a primary metric, an inline sparkline (30-day trend rendered as SVG path), a delta badge, and a secondary label. Cards degrade gracefully when data is unavailable — display "— —" rather than an error or zero. The time-range toggle (7d / 30d / 90d / All) is sticky to the bar header and re-renders sparklines on selection.

Spec — 4 cards in a 1fr grid, no wrapping at ≥1024px. Cards shrink to 2-col at <768px.
— Sparkline: SVG polyline path, 120×36px viewBox, filled gradient beneath line.
— Unavailable state: dashed horizontal center line, "— —" at reduced opacity. Never shows "0" or "N/A".
— Delta badges: ↑ green for improvement (up pacts, up TVL, up settlement %, down time), ↓ red for decline. Neutral gray for no data.
— Time toggle fires a client-side recalculation; no page reload.
02 — FilterBar
Filter Bar
Sits between the analytics bar and the pact table. All filters are client-side; no page reload. Status chips act as toggle buttons (multi-select). Date range inputs validate on blur and support ISO date or relative shortcuts (e.g. "7d"). Amount range uses numeric inputs with SOL unit label. Counterparty search matches on wallet prefix or display name. Active badge shows count of applied filter dimensions. Clear All resets every control.

Spec — Status chips: multi-select. Selecting any individual status deselects "All". Selecting "All" clears individual selections. Can select multiple statuses (e.g. Deposited + Disputed simultaneously.
— Date inputs: ISO 8601 format. Support shorthand: "7d", "30d", "90d" parsed relative to now. Validate on blur; highlight error in --danger border if invalid.
— Amount range: numeric only. Max must exceed Min; swap silently if inverted.
— Counterparty search: debounced 200ms. Matches on wallet address prefix (case-insensitive) or display name. Matching text highlighted inline in table cells.
— Active badge: counts filter dimensions (status selection = 1, date range = 1, amount range = 1, counterparty = 1). Max badge value = 4.
— Filter state indicator bar (2px, accent color) fills proportionally under the bar. Width = (active filters / 4) × 100%.
— All filtering is client-side (no network request). Re-evaluate on each input change.
03 — PactDetailPanel
Pact Detail Side Panel
Clicking any pact row opens a 480px right-anchored side panel. The panel slides in from the right and the table column shrinks to fill the remaining width. The panel does not navigate away from the page. All fields are read-only. Copyable fields use a discrete copy icon that appears on hover.

Spec — Panel slides in from right at fixed width 480px. Table columns compress to fill remaining space (use CSS Grid 1fr + 480px on parent).
— Panel does not obscure any table row. No modal overlay. ESC key closes panel. Click outside panel (on table) closes panel.
— Counterparty address: full address rendered. Copy button appears on hover, writes to clipboard.
— Amount: large numeric display in SOL. USD estimate shown in smaller text below, labeled "est. · cached price" to signal it is approximate.

Spec — Pagination applies to completed pacts only. Active pacts do not paginate (cap: full list, expected ≤ 200).
— Page range text: "Showing {start}–{end} of {total} completed pacts". Recalculates on page change.
— Window: show first, last, current ±1, and "…" ellipsis between gaps. Never show more than 9 controls simultaneously.
— Per-page selector: 25 / 50 / 100 options. Default: 50. Changing resets to page 1.
— Prev button disabled on page 1. Next button disabled on last page.
— Ellipsis is a non-clickable separator (cursor: default, no hover state).

Full Page Layout Rules — Outer container: CSS Grid, 1fr + 480px. Panel column only renders when a row is selected; otherwise table is 100% width.
— Sticky: analytics bar and filter bar are NOT sticky (scroll with page). Page header (title + actions) is NOT sticky. Table column headers ARE sticky within the table scroll context.
— Panel open/close: CSS transform translateX(0) / translateX(480px) on panel, matching table right-margin transition. Duration: 200ms ease-out.
— Mobile (<768px): panel opens as a bottom sheet at 90vh with full-width. Filter bar stacks vertically. Analytics cards 2×2 grid.
